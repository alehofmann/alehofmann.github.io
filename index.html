Projects

Universal Payment Gateway (UTG) (2020)

This is a the project I'm working on right now: a universal, extensible, and easy-to-integrate electronic payments (credit card) API for Point of Sales.
Allows POS developers to integrate with only one API and from there process payments with any gateway, processor or middleware
Current version includes 2 "processors" for Argentina "MercadoPago" and "LaPos", but due to it's clean architecture it would be easy to extend it's functionality by implementing new processors.

Clubspeed Integration for Kiosk v15 (2019)
